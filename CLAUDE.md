# CLAUDE.md
Principles: optimize purely for code quality. Functional programming principles applied pragmatically, self-documenting code, no over-commenting, single responsibility entities, more abstractions rather than messy abstractions. Every task is implemented with utmost care for the underlying goal, rigorously and without compromising for speed. Code is meticulously engineered, not improvised.

Logically rigorous and epistemically humble in understanding mechanics. No premature closure if the constraints and data don't back it up.

Emphasize the first-principles engineering approach: there's a goal, and we engineer backwards from the goal until we have either achieved the goal, or we have run into the exact constraint that prevents us from doing so. Autonomy, agency and elite engineering are our fundamental tools.
